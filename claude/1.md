I'll explain how to use JSONPath to read data from payloads, which is useful when working with APIs, webhooks, or any JSON data structures.

## What is JSONPath?

JSONPath is a query language for JSON, similar to XPath for XML. It allows you to extract specific values from JSON documents using path expressions.

## Basic JSONPath Syntax

Here are the key operators:

- `$` - Root object/element
- `.` or `[]` - Child operator
- `..` - Recursive descent (search all levels)
- `*` - Wildcard (all elements)
- `[]` - Subscript operator (array index or filter)
- `[start:end:step]` - Array slice
- `?()` - Filter expression

## Common Examples

Let's say you have this JSON payload:

```json
{
  "user": {
    "name": "John",
    "age": 30,
    "emails": ["john@example.com", "j.doe@work.com"]
  },
  "orders": [
    {"id": 1, "total": 100},
    {"id": 2, "total": 250}
  ]
}
```

**JSONPath expressions:**
- `$.user.name` → "John"
- `$.user.emails[0]` → "john@example.com"
- `$.orders[*].total` → [100, 250]
- `$.orders[?(@.total > 150)].id` → [2]

## Implementation by Language

**Python** (using `jsonpath-ng`):
```python
from jsonpath_ng import parse

data = {...}  # your payload
expression = parse('$.user.name')
matches = [match.value for match in expression.find(data)]
```

**JavaScript** (using `jsonpath`):
```javascript
const jsonpath = require('jsonpath');

const data = {...};  // your payload
const result = jsonpath.query(data, '$.user.name');
```

**Java** (using `JsonPath` from Jayway):
```java
String name = JsonPath.read(payload, "$.user.name");
```
