## 🔥 Exam “Tell” (How to spot the correct answer fast)

If the question says **“as accurate as possible”** and involves **chargeback/showback across multiple accounts**, the best answer is almost always:

> ✅ **Cost Allocation Tags + AWS Cost & Usage Report (CUR)** (often with **Athena**)

***

## ✅ One-liner (Exam-ready)

**Correct:** *Activate `costCenter` as a cost allocation tag and use CUR with resource IDs + hourly granularity; query CUR (Athena) to sum costs for `costCenter=compliance` and charge back to the compliance account.*
