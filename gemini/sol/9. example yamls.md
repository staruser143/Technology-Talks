Below are **10 fully-worked example YAML mapping specifications**, each demonstrating a different real-world scenario for extracting & transforming JSON into **flattened AcroForm fields** using the mapping engine.

Each YAML follows the structure:

```yaml
template:
  file: ...
  location: ...

fields:
  targetPdfFieldName:
    expr: "..."             # JSONPath or JSONata
    transform: [ ... ]      # optional list
```

# ‚úÖ **Example 6 ‚Äî Extract Phone Number with Fallback Logic**

Pick mobile if present, else home:

```yaml
template:
  file: "contact_form.pdf"
  location: "classpath:/forms"

fields:
  phone:
    expr: "$.phones[?(@.type=='MOBILE')].number"
    transform: ["fallbackIfEmpty:$.phones[?(@.type=='HOME')].number"]
```

(*Assumes a custom `fallbackIfEmpty` transform*)
